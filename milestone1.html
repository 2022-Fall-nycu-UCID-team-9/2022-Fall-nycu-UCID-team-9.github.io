1-1

Problem Statement

長者在就醫看病時，常常無法與醫生準確表達自己身體狀況、平時的生活情況。同時，許多年長患者因為是慢性疾病，必須吃許多不同種類，卻又相似的藥物。因而，在事後用藥時，這樣複雜的情況有時便會造成老人誤食、重複用藥或忘記用藥。
我們由上述問題發想——藉由開發一個同步更新就醫紀錄、醫囑、用藥須知等資訊的應用程式，讓年長患者親屬也能透過該應用程式介面，填寫病患身體狀況、查看醫囑等，作為醫生、年長者及其親屬的資訊匯集地，以改善老人就醫時所遇到的困擾與潛在錯誤用藥情形造成的身體危害。

Target Population

年長患者及其親屬

Importance

許多年長者就醫時，會發生與醫生溝通不良的情況。此時，醫生無法準確瞭解老人的身體情況及日常生活作息。且在慢性病用藥醫囑複雜的情況下，常有年長慢性病患者不當用藥的情形發生。根據健保資料庫表明，每年65%年長者曾發生一次潛在不當用藥的狀況。

Stakeholder

1. 年長患者
2. 年長患者家屬
3. 醫院
4. 醫生

1-2

Problem Statement

在聯合國提出的永續發展目標中(target3.4)提出：在西元2030年前，消除可預防的新生兒以及五歲以下兒童的死亡率。根據衛生福利部統計處110年兒童死亡原因統計的結果，前五大死因依序是：源於周產期的特定病況、先天性畸形變形及染色體異常、事故傷害、癌症、嬰兒猝死症候群。其中，屬於可預防的死因有事故傷害和嬰兒猝死症候群。然而，現今社會的家庭結構多是雙薪家庭，在上班期間難以照顧嬰兒，即使請保母、托兒所或祖父母代為照顧，疏於照顧導致嬰兒意外死亡的新聞仍層出不窮，更別提睡覺時間。此時若有個能24/7確保嬰兒周遭環境、生命徵象和睡覺姿勢的應用程式便能減低事故傷害和嬰兒猝死症發生的機率。

Target Population

雙薪家庭且家中有5歲前的嬰兒的家庭

Importance

如今少子化趨勢越來越嚴重，雖然已有許多激勵生育的措施，但若生出來的小孩沒辦法得到妥善照顧而死亡，就猶如沒堵住排水孔的浴缸。並且，根據世界衛生組織統計，台灣至4歲兒童死亡率高達千分之2.7，甚至輸給許多發展中國家，因此有效預防嬰兒死亡的措施對現今的台灣來說非常重要。

Stakeholder

1. 缺乏時間和人力照顧嬰兒的家庭
2. 想要確保嬰兒安全的父母

1-3

Problem Statement

衣櫃裡總有許多閒置許久的衣服，可能是因為尺寸不合、衝動消費或是因「快時尚」風潮而買的服飾，綠色和平組織的台灣消費習慣與心理調查裡提到，20-45歲的民眾容易因衝動購物而購買衣服，54%的民眾曾經因為促銷活動，買下不需要的東西；近四分之一的民眾（24%），因為名人代言而衝動購買不需要或不適合的商品。超過半數的民眾（54%）表示擁有的衣服比實際需要的多；超過七成（72%）的民眾曾經在衣櫃裡發現標籤未拆的衣服。在這種消費模式下產生了過度消費、環境破壞、勞工權益等等的問題。

Target Population

1.容易衝動購物的人
2.時常在更換衣服的人

Importance

在生產衣服的過程，會消耗大量的自然資源，以及破壞環境生態，根據聯合國永續發展目標第12項，我們應該具有永續發展的相關資訊和意識，在這種購物模式下，尋找友善環境的解方。

Stakeholder

1. 容易買到不需要的衣服、衝動購物的人
2. 想以低價購買衣服的人
3. 想找到適合的衣服的人
